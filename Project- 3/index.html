LITURATURE SHOWCASE

INTRODUCTION

While searching for a book to use for this project I came across this book of poems by Jana Prikryl. Some short, some long, all that I read were beautiful. What I really liked about her work was learning about where she came from. Her backstory is so interensting and so impressive to learn that she is now working for such a large publication. My favorite poem that I read was TO TELL OF BODIES CHANGED which I have inclued in the sample page.
